# Jovo Plugins

Plugins allow you to easily extend the Jovo Framework without having to mess with its core code and architecture. Check out the [documentation](https://www.jovo.tech/docs/advanced#plugins) to learn how to create your own plugins.

# List of Plugins
To add your plugin to the table below, fork the repo and add your plugin to the `plugins.json` file in the same format as the other. After that run the `npm run docs` script as well as commit and push your changes. At the end, open a pull request with the changes and we will do the rest.
